# APA ITU GITHUB?

![gambar](./images/thumbnail-8.jpg)

***GitHub*** adalah platform berbasis cloud untuk menyimpan repositori Git.
Platform ini memudahkan developer untuk berkolaborasi, mengelola proyek, serta melacak perubahan kode secara online.

Berikut adalah poin-poin penting tentang GitHub:
| Aspek | Penjelasan |
|--- | --- |
|Hosting Kode & Git | Tempat menyimpan repositori Git agar perubahan kode bisa dicatat dan dilihat dengan mudah dan efisien.
| Kolaborasi Tim | Ada fitur seperti pull request, code review, dan issue tracking supaya tim bisa bekerja bersama tanpa menghancurkan atau mengacaukan kode satu sma lain. |
| Portofolio Developer | Pengguna bisa menampilkan proyeknya dan terhubung dengan developer lain. |
| Fitur Utama | Mendukung version control, kerja  online bersama, integrasi CI/CD, dan wiki untuk dokumentasi. |

> ### Bagaimana Cara Memulai GitHub?
> Ketika kita masih baru mengenal atau mengetahui tentang git kita dapat memulai dengan membuat akun baru lalu mempelajari dasar dasar github dan kunci dalam berkolaborasi

### Dalam WorkFlow Berbasis Git
Biasanya dalam WorkFlow berbasis Git kita akan :
- Membuat branch dari file utama yang sedang dikerjakan bersama tim.
- Mengedit file secara mandiri dan aman di branch milikmu sendiri.
- Membiarkan Git menggabungkan perubahanmu ke file utama tanpa merusak perubahan orang lain.
- Membiarkan Git mencatat semua perubahan agar semua orang tetap menggunakan versi terbaru proyek.

Kalau ingin mencoba Git sendiri, lihat panduan [Getting started with Git](https://docs.github.com/en/get-started/learning-to-code/getting-started-with-git).
